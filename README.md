## 📖 Overview
DignityGuard AI is a prototype system designed to protect human dignity in the AI era.  
It demonstrates a **three-agent workflow**:
- **Scout Agent** – Detects misuse in requests.
- **Guardian Agent** – Verifies explicit consent before proceeding.
- **Hunter Agent** – Escalates cases to human moderators.

---

## 🔑 Features
- **Login page** (`login.html`) – Users must log in before accessing the app demo.  
- **App demo** (`app.html`) – Runs the Scout → Guardian → Hunter workflow.  
- **Consent log** – Records every decision with timestamp and allows CSV download.  
- **Homepage** (`index.html`) – Introduces the agents and links to login/app demo.
